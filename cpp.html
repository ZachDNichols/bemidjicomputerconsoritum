What is C++?

C++ is a general-purpose programming language created by Danish computer scientist Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".
Source: Wikipedia

What is C++ used for?

C++ is used for a variety of many modern applications. C++ is even used in the web browser you are viewing this on! C++ is also used for things like game engines, proffesional applications such as the Adobe suite, or even operating systems!
